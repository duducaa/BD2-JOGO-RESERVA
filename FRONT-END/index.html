Descrição do Projeto

A “AVENTURA NA FLORESTA ENCANTADA: O ENIGMA DO BANQUETE REAL”, é um projeto em desenvolvimento para auxiliar no aprendizado das consultas em um banco de dados, Este projeto foi inspirado no SQL Murder Mystery, e nele jogador assumirá o papel do detetive James Constantine Bugs, investigando o misterioso assassinato ocorrido durante o banquete real do casamento na Floresta Encantada. Nesse caso, o jogador deverá utilizar as habilidades em SQL para vasculhar tabela por tabela na base de dados até descobrir o verdadeiro assassino. Nesse jogo, você terá a ajuda do seu companheiro de trabalho Harley, um fantasma que apenas aparece para você e lhe ajuda quando lhe é conveniente. No jogo você precisa avaliar cada um dos depoimentos, seguir seus instintos e decidir quem é suspeito.

ÍNICIO

A história se passa em um dia lindo numa floresta, onde haverá o casamento da filha da rainha do reino mágico de Aurora. Para o casamento todos os lideres de cada clã foram prestigiados com um convite, mas o que era para ser um dia perfeito para os noivos se tornou um grande mistério de assassinato, onde só você pode desvendar tudo e encontrar o culpado.

Objetivo

O objetivo final é como um jogo de adivinhação. Você precisa descobrir quem é o culpado e provar isso com o ID único do suspeito. É como um jogo de detetive, mas em vez de procurar pistas em um tabuleiro, você está procurando pistas em um banco de dados!

Comandos SQL Básicos

SELECT

Este comando é usado para selecionar dados de um banco de dados. Os dados retornados são armazenados em uma tabela de resultados.

SELECT column1, column2 FROM table_name;

UPDATE

O comando UPDATE é usado para alterar os dados existentes em uma tabela.

UPDATE table_name SET column1 = value1, column2 = value2 WHERE condition;

DELETE

O comando DELETE é usado para deletar registros existentes em uma tabela.

DELETE FROM table_name WHERE condition;

INSERT

O comando INSERT INTO é usado para inserir novos registros em uma tabela.

INSERT INTO table_name (column1, column2) VALUES (value1, value2);

JOIN

O comando JOIN é usado para combinar linhas de duas ou mais tabelas, com base em uma coluna relacionada entre elas.

SELECT column_name(s) FROM table1 JOIN table2 ON table1.column_name = table2.column_name;

WHERE

O comando WHERE é usado para filtrar registros.

SELECT column1, column2 FROM table_name WHERE condition;
